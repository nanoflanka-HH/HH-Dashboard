# data

此資料夾放 Dashboard 可直接讀取的基礎資料。

## 目前主要資料檔

- `abg-base-headcount.csv`：ABG / A事業群基礎人力資訊，Dashboard 預設會自動讀取此檔。

## 建議欄位格式

```csv
日期,總人數,IDL,DL,派遣
20260105,371669,52320,155447,163902
20260331,341223,51151,146183,143889
20260630,321566,51222,147767,122577
```

## 更新方式

若每月或每季要更新，只要覆蓋 `abg-base-headcount.csv`，並保留同樣欄位名稱即可。
Dashboard 會自動計算：

```text
正職 = IDL + DL
總計 = 正職 + 派遣
```
